Escape Game virtuel conçu pour une foire aux clubs informatique. Les joueurs doivent infiltrer un serveur sécurisé en résolvant 4 énigmes progressives.

## 🚀 Accès en ligne

Scannez ce QR Code pour accéder au jeu :

![QR Code](qr-code.png)

Ou rendez-vous sur : https://github.com/melinoses/escape-game.git

## 📖 Histoire

Le joueur est une nouvelle recrue qui doit infiltrer le serveur sécurisé d'une organisation mystérieuse pour récupérer un fichier "Top Secret".

## 🎯 4 Niveaux

1. **Terminal** - Navigation dans un système de fichiers
2. **Cryptographie** - Chiffrement de César à décoder
3. **Analyse de données** - Trouver l'anomalie dans une grille
4. **Portes logiques** - Activer les bons interrupteurs (AND, OR, NOT)

## ⏱️ Durée

30-35 minutes maximum (timer intégré)

## 📱 Compatible

- Smartphones (optimisé tactile)
- Tablettes
- Desktop

## 🛠️ Déploiement local

Ouvrez simplement `escape-game.html` dans un navigateur.

## 🔧 Personnalisation

Modifiez la section `CONFIG` dans le JavaScript pour ajuster :
- Les solutions des énigmes
- Le temps maximum
- Le code final
