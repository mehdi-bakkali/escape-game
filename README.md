# 🎮 Infiltration 2026 - Virtual Escape Game

Virtual Escape Game designed for Addictech. Players must infiltrate a secure server by solving 4 progressive puzzles.

## 🚀 Online Access

Visit to access the game:: 
**https://mehdi-bakkali.github.io/escape-game/escape-game.html**

## 📖 Story

You are a new recruit who must infiltrate a mysterious organization's secure server to recover a "Top Secret" file.

## 🎯 4 Levels

1. **Terminal** - Navigate through a file system using commands
2. **Cryptography** - Decode a Caesar cipher message
3. **Data Analysis** - Find the anomaly in a data grid
4. **Logic Gates** - Activate the right switches (AND, OR, NOT)

## ⏱️ Duration

30-35 minutes maximum (built-in timer)

## 📱 Compatible

- Smartphones (touch optimized)
- Tablets
- Desktop

## 🛠️ Local Deployment

Simply open `escape-game.html` in a browser.

## 🔧 Customization

Edit the `CONFIG` section in the JavaScript to adjust:
- Puzzle solutions
- Maximum time
- Final code
