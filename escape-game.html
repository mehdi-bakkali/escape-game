🔐 BRIEFING DE MISSION

Bienvenue, nouvelle recrue.

Notre organisation a intercepté des communications provenant d'un serveur hautement sécurisé. Votre mission : infiltrer le système, contourner les protocoles de sécurité et récupérer le fichier "TOP_SECRET.dat".

Quatre couches de sécurité vous attendent. Chaque énigme résolue vous rapproche du cœur du système.

⚠️ ATTENTION : Le système dispose d'un protocole d'auto-destruction. Vous avez 35 minutes maximum avant que toutes les données ne soient effacées.

💻 COUCHE 1 : LIGNE DE COMMANDE

Vous avez accès à un terminal distant. Le premier code se trouve dans un fichier caché quelque part dans le système.

Commandes disponibles : ls, cd [dir], cat [file], pwd

Connection établie au serveur distant...
Système: SecureOS v4.2
Entrez vos commandes ci-dessous :
$
📍 Indice 1 : Commencez par explorer le répertoire actuel avec ls, puis naviguez vers /secure
📍 Indice 2 : Le fichier secret s'appelle access_code.txt et se trouve dans /secure/classified

🔐 COUCHE 2 : MESSAGE CHIFFRÉ

Message intercepté sur le réseau interne :

Message original :
Décalage: 0
Message décodé :

Le mot de passe est le message décodé (sans espaces, en majuscules).

📍 Indice 1 : C'est un chiffrement de César. Essayez différents décalages jusqu'à ce que le texte ait du sens.
📍 Indice 2 : Le mot décodé est un terme informatique courant en 4 lettres.

📊 COUCHE 3 : ANOMALIE DE DONNÉES

Base de données sismique corrompue. Une valeur ne suit pas la séquence logique.

Trouvez l'anomalie et entrez sa position (ligne-colonne) comme mot de passe (ex: 2-3).

💡 Cliquez sur la valeur suspecte pour la sélectionner, puis validez.

📍 Indice 1 : Chaque ligne suit une progression mathématique simple. Cherchez le nombre qui brise le motif.
📍 Indice 2 : Ligne 1: +2, Ligne 2: +3, Ligne 3: ×2, Ligne 4: nombres premiers. L'anomalie est en position 2-4.

⚡ COUCHE 4 : PORTES LOGIQUES

Activez les bons interrupteurs pour que le courant atteigne la sortie finale.

Configuration requise : Toutes les sorties doivent être ACTIVES (vertes)

AND
OR
NOT
FINAL OR

💡 AND = les 2 doivent être ON | OR = au moins 1 ON | NOT = inverse l'état

📍 Indice 1 : Pour AND, activez les deux interrupteurs. Pour OR, un seul suffit. Pour NOT, laissez éteint.
📍 Indice 2 : Ligne 1: A1+B1 ON | Ligne 2: A2 ou B2 ON | Ligne 3: A3 OFF
